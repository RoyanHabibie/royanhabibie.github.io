The first thing you should learn in web programming is to use html.
HTML is the language used to create web pages.
For those who work in the world of gadgets and computer science, you must be familiar with the various codes in it.
HTML is short for Hypertext Markup Language.

After that you can try css. CSS stands for cascading style sheets, which is the language used to determine the appearance and format of web pages. With CSS, you can set the font type, text color, and page background.
CSS is used in conjunction with markup languages to build an attractive and functional website that works

Then there is JavaScript which is a programming language used in website development to make it more dynamic and interactive. If previously you only knew HTML and CSS, now you know that JavaScript can increase the functionality of web pages. Even with this JavaScript you can create applications, tools, or even games on the web.
